# URL Shortener API

Сервис сокращения ссылок с JWT-аутентификацией и статистикой переходов. Создан в учебных целях с использованием AI-ассистента.

## Функционал
- Регистрация и вход пользователей (JWT-токены)
- Создание коротких ссылок
- Редирект на оригинальный URL с подсчётом кликов
- Статистика по своим ссылкам (количество переходов, дата создания)

## Технологии
- Python 3.14.5
- FastAPI
- SQLite
- Passlib (sha256_crypt)
- python-jose (JWT)
