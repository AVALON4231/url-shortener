# URL Shortener

![Tests](https://github.com/AVALON4231/url-shortener/actions/workflows/tests.yml/badge.svg)

Персональный сервис сокращения ссылок с JWT-аутентификацией, автоматическим извлечением заголовков страниц и персональной историей. Доступен через веб-интерфейс и API (Swagger).

Проект создан в учебных целях с использованием AI-ассистента, но решает реальные задачи: помогает организовать закладки, отслеживать переходы и делиться ссылками без лишнего шума.

## Ключевые особенности

- **Приватность «из коробки»** — каждый пользователь видит только свои ссылки и статистику, как в коммерческих сервисах.
- **Умное сохранение контекста** — сервис автоматически извлекает заголовок страницы (`<title>`) и показывает его в истории вместо голого URL.
- **Адаптивный ответ** — редирект для браузеров и JSON для API-клиентов (в зависимости от заголовка `Accept`).
- **Гибкое хранилище** — автоматически переключается между SQLite (локально) и PostgreSQL (на продакшене) без изменения кода.
- **Полный цикл CI/CD** — тесты через GitHub Actions, деплой на Render, документация Swagger с возможностью авторизации.
- **Продуманный UX** — регистрация сразу переводит к сокращению ссылок, история загружается с сервера, пароль можно посмотреть при вводе.

## Функционал

- Регистрация и вход по любому логину (JWT-токены)
- Создание коротких ссылок (принадлежат конкретному пользователю)
- Редирект на оригинальный URL с подсчётом кликов
- Статистика по своим ссылкам (клики, дата создания)
- Возврат JSON для Swagger и редирект для браузеров
- Автоматическая документация API (Swagger)
- Современный веб-интерфейс на Bootstrap 5 (регистрация → вход → сокращение)

## Технологии

- Python 3.11+
- FastAPI
- PostgreSQL (production), SQLite (разработка)
- Passlib (sha256_crypt)
- python-jose (JWT)
- aiohttp (для асинхронного получения заголовков)
- pytest + моки и dependency overrides
- GitHub Actions (CI)
- Bootstrap 5 (фронтенд)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/AVALON4231/url-shortener.git
   cd url-shortener
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate      # Linux/Mac
   venv\Scripts\activate         # Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Задайте переменные окружения (опционально):
   - `SECRET_KEY` — случайная строка для подписи JWT
   - `DATABASE_URL` — строка подключения к PostgreSQL (если не задана, используется SQLite)

5. Запустите сервер:
   ```bash
   uvicorn main:app --reload
   ```

После запуска:
- Веб-интерфейс: [http://127.0.0.1:8000](http://127.0.0.1:8000)
- Документация API: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

## Тестирование

```bash
pytest
```

Тесты прогоняются автоматически при каждом пуше через GitHub Actions.

## Деплой

Сервис развёрнут на [Render](https://render.com) с автоматическим деплоем из ветки `main`.
